## Course Curriculum

### Module 1: Introduction to Retrieval-Augmented Generation (RAG)

**Concepts Covered**:
- Understanding RAG: combining information retrieval with generative AI
- How RAG produces accurate, context-aware responses
- Core principles and architecture

**Hands-on Lab 1**:
- Use LangChain to split and embed documents
- Construct simple retrieval chains
- Work within Jupyter Notebook environment
- Build foundational RAG skills

### Module 2: Building a RAG Application

**Integration & Development**:
- Transform RAG knowledge into working applications
- Integrate LangChain with large language models (LLMs)
- Develop applications using LangChain with Gradio
- Enable real-time interaction with AI models

**Project Outcome**:
By the end of this module, you will have built a question-and-answer bot that:
- Retrieves answers from loaded documents
- Demonstrates the complete RAG pipeline from data to interface
- Provides hands-on experience with production-ready applications

### Module 3: Alternative Frameworks with LlamaIndex

**Framework Exploration**:
- Learn LlamaIndex as an alternative to LangChain
- Transfer RAG skills across different frameworks
- Build an icebreaker bot for meaningful conversations
- Work with IBM Granite and LlamaIndex

**Technical Components**:
- Vector databases
- Embedding models
- Chunking strategies
- Retrievers
- Prompt templates
